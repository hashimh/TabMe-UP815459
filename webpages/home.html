creating tabs

in order to create a tablature, first choose the type of guitar part in the stave options section and click create. notes can now be inserted using the fretboard, and chords from the chord selection section.

creating chords

custom chords that aren't included in the pre-saved list can be created with the create chord section on the right. choose the starting position of the chord, then select the notes on each string to form the chord shape, name it, and save it. it will be available to you for use now and in future sessions in the 'my chords' list.

tab symbols

the symbols section allows the use of the main symbols used universally in tablature format. these will only be placed if used correctly, i.e. a hammer on must be on the same string, and on a higher fret than its predecessor:

  • h - hammer on
  • b - bend
  • p - pull off
  • / - slide up
  • \ - slide down
  • ~ - vibrato
