Creating a Tablature

In order to create a tablature using Tabify, you must first select the type of instrument from the Stave Creation section. Upon creating a stave, use the main fretboard for lead parts, or the chord section the right hand side to create or select chords for rhythm parts.
You can also load in previous tablatures using the 'Load' button on the navigation bar.

Creating Chords

Use the miniature fretboard on the right side of the form to create chords not included in the 'presaved' list. Simply choose the starting position of the chord, and select the frets of the chord on the specific strings.
Name the chord, and click the save button for it to be available for you during any session, in the 'My Chords' list.

Tab Symbols

The 'Tab Options and Symbols' section provides the key symbols used universally in the tablature format:
h - Hammer On
b - Bend
p - Pull Off
/ - Slide Up
\ - Slide Down
~ - Vibrato
